# Responsible Use

`Metal_Mnemonic_Recovery` is intended for lawful mnemonic recovery, backup validation, and authorized audit workflows.

Use it only when:

- the wallet or mnemonic belongs to you, or
- you have explicit authorization to recover it

Do not use this project for:

- unauthorized wallet access
- credential theft
- scanning third-party data without permission
- bypassing ownership or custody controls

The repository is maintained as a recovery tool, not as an intrusion tool.
