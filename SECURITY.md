# Security Policy

If you believe you found a security issue in `Metal_Mnemonic_Recovery`, do not open a public issue first.

Report it privately to the repository owner and include:

- affected version or commit
- reproduction steps
- expected and actual behavior
- whether the issue affects correctness, secrecy, integrity, or release packaging

This project is intended only for lawful recovery of wallets, backups, and seed phrases that belong to you or that you are explicitly authorized to recover.
